## Alt Text Guidelines (from Vol 1)

1. **Brief description** (1-2 sentences) for screen readers
2. **Mathematical content**: Include key formulas/values shown
3. **Structural info**: Describe layout (left/right panels, labels)
4. **No "image of"** — just describe what it shows
5. **For graphs**: Include axes, curves, key points, shading

### Example (from Vol 1):

```
"Graph showing f(x) = x² with a secant line connecting points 
(a, f(a)) and (b, f(b)). As b approaches a, the secant line 
approaches the tangent line at (a, f(a))."
```
