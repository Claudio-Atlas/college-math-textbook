## Issues Found

### 1. [CRITICAL] Custom LaTeX Macros Not Defined in MathJax

**Affected sections:** All 13 sections in Chapters 5-6

**Problem:** The content JSON files use custom LaTeX commands that are not defined in the MathJax configuration (`src/components/MathJaxProvider.tsx`). This causes these commands to render as raw text instead of proper math notation.

**Missing macros and usage count:**
| Macro | Usage Count | Purpose |
|-------|-------------|---------|
| `\dx` | 321 | Differential dx (∫...dx) |
| `\dt` | 64 | Differential dt |
| `\defint` | 34 | Definite integral with bounds |
| `\deriv` | 35 | Derivative notation |

**Current MathJax macros defined:**
```javascript
macros: {
  abs: ['\\left|#1\\right|', 1],
  R: '\\mathbb{R}',
  N: '\\mathbb{N}',
  Z: '\\mathbb{Z}',
  Q: '\\mathbb{Q}',
  C: '\\mathbb{C}',
  vec: ['\\mathbf{#1}', 1],
  dd: ['\\,\\mathrm{d}#1', 1],
  dv: ['\\frac{\\mathrm{d}#1}{\\mathrm{d}#2}', 2],
  pdv: ['\\frac{\\partial #1}{\\partial #2}', 2],
}
```

**Recommended fix:** Add these macros to `MathJaxProvider.tsx`:
```javascript
macros: {
  // ... existing macros ...
  dx: '\\,\\mathrm{d}x',
  dt: '\\,\\mathrm{d}t',
  du: '\\,\\mathrm{d}u',
  defint: ['\\int_{#1}^{#2}', 2],
  deriv: ['\\frac{\\mathrm{d}#1}{\\mathrm{d}#2}', 2],
}
```

**Visual impact:** In Section 6.5 (The Substitution Rule), the opening equation displays as:
```
\defint{a}{b} f(x)\dx = F(b) - F(a)
```
instead of the properly rendered integral notation.
